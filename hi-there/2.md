文件系统权限

macOS 系统中，运行 `ls -al` 命令可以查看文件列表以及权限相关的一些信息

以 `drwxr-xr-x` 为例，第一个字符 `d` 表示 `directory`，剩余 9 个字符可以拆分成以下三个部分

|字符串|十进制数字|名称|简写|
|-|:-:|-|:-:|
|rwx|4 + 2 + 1 = 7|User|u|
|r-x|4 + 0 + 1 = 5|Group|g|
|r-x|4 + 0 + 1 = 5|Others|o|

可以通过 `chmod` 命令改变权限

```
chmod a+r
chmod u+x
chmod g+x
chmod o-r
chmod a-x
```

> 多数情况下，`a` 表示 `all`
